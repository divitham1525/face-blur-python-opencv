# Face Blur System
## About
   This project detects faces in images and applies blur to them using Python and OpenCV.
   
## How to Use
1. Clone the repository  
2. Install required libraries  
3. Run the script  

## Features
 Face detection  
 Face blurring  
 Simple command line usage
